User trust is a core principle of Orbit.

**What Orbit Accesses**

- Microphone (only when voice command is enabled)
- System controls explicitly approved by the user

**What Orbit Never Does**

- No background or continuous recording
- No permanent storage of voice data
-  No sharing of voice or system data with third **parties**

All permissions are transparent and revocable.
